# Set Hotbars
A set of three macros for quickly setting the basic hotbars

## Hotbar 1
```
/micon "Soul of the Sage" item
/hotbar action Dosis 1 1
/hotbar action Phlegma 1 2
/hotbar action Dyskrasia 1 3
/hotbar action Toxikon 1 4
/hotbar action Pneuma 1 5
/hotbar action Psyche 1 6
/hotbar action Esuna 1 9
/hotbar action Rescue 1 10
/hotbar action Egeiro 1 11
/hotbar action Swiftcast 1 12
/hotbar display 1 On
```

## Hotbar 2

/micon "Soul of the Sage" item
